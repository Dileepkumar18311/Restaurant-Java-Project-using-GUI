MyRestaurant Application
This is a simple restaurant and coffee ordering application built using Java Swing. The application allows users to enter their customer details, select their food and drink items, and choose their preferred type of drink. Users can then generate a receipt and print it out.

Installation
To run the application, you need to have Java installed on your computer. You can download and install the latest version of Java from the official website.

Once you have installed Java, you can download the application source code and compile it using a Java IDE, such as Eclipse or NetBeans. Alternatively, you can run the pre-compiled executable JAR file from the command line using the following command:

java -jar MyResturant.jar

Usage
To use the application, follow the instructions on the GUI. You can enter your customer details in the text fields, select your food and drink items from the dropdown lists, and choose your preferred type of drink using the radio buttons. Once you have made your selections, click the "Receipt" button to generate a receipt. You can then click the "Print" button to print the receipt.
